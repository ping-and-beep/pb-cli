# Usage
<!-- usage -->
```sh-session
$ npm install -g pingbeep-cli
$ pingbeep COMMAND
running command...
$ pingbeep (-v|--version|version)
pingbeep-cli/0.1.0 darwin-x64 node-v12.16.1
$ pingbeep --help [COMMAND]
USAGE
  $ pingbeep COMMAND
...
```
